Welcome to "The MoSquad App". This app lets the user choose a service package 
offered by Mosquito Squad. The user first creates a new account and logs in
Then started a new estimate they can enter their address. After that
they will be able to set the treatment boundaries of their property using 
a google map view. Next they will choose the treatment package of their choice.
Continuing the user will then be able to see the overview of their estimate.
On the home screen the user will be able to see a list of all their estimates.
Along with access to package details and user reviews with a click of a button.
